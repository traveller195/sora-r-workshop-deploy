hamburg {sora}R Documentation

Structural dataset for Hamburg, Germany

Format

A dataframe with 12,618 rows and 4 columns: crs = 4647

id

Character vector containing fake IDs demonstrating what real survey IDs could look like

syear

Year of the survey for the specific household.

x

x coordinate

y

y coordinate

Source

Data is based on the SOEP data v38 and contains data for the years 2000 until 2021. Coordinates for the households in SOEP starting in 2000. That dataset 'hamburg' is a subset of the 'structural dataset' which can be requested by SOEP. This 'structural dataset' concerns synthetic coordinates for the SOEP househols, which are based on real coordinates for houses. That dataset for the coordinates is provided by 'BKG'.
